Node.Js

"Es una plataforma basada en el motor JavaScript de Chrome para la construcción fácil y rápida de aplicaciones de red escalables. Node.js utiliza un event-driven, modelo sin bloqueo de E / S que hace que sea ligero y eficiente, ideal para aplicaciones de uso intensivo de datos en tiempo real que se ejecutan a través de dispositivos distribuidos."
